# Contributing to EduConnect

Thank you for considering contributing to EduConnect! Whether you're sharing a resource, adding a reflection, or participating in discussions, your input is valuable to our community.

## How to Contribute

### 1. Reporting Issues
If you find any bugs or have suggestions, please open an issue using the template provided in `.github/ISSUE_TEMPLATE/`.

### 2. Adding to Discussions
You can add to or start a new discussion in the `community/discussions/` folder by editing or creating a Markdown file.

### 3. Sharing Resources
To share lesson plans, teaching strategies, or creative activities:
- Navigate to `community/resources/` and find the appropriate folder.
- Add your resource in Markdown format.

### 4. Submitting a Pull Request
- Ensure your changes are consistent with the project's structure and guidelines.
- Create a pull request with a detailed description of your changes.

### 5. Reflective Journal Contributions
- Create a new folder under `reflective_journal/user_journals/` with your GitHub username.
- Add your reflective journal entries to this folder.

## Code of Conduct
Please read and adhere to our [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming and respectful community.
